# RTAL-EAI-004-C64-SFX-Sound-Sampler-Advancement
The Commodore SFX Sound Sampler is a rare hardware cartridge released by Commodore UK in the 1980s. It allowed users to convert analog audio into digital data to be stored, modified, and played back directly on the ⁠Commodore 64. 

Here, it is being redeveloped on an ESP32, no C64 and SFX needed.

Work in Progress - further Information soon.
